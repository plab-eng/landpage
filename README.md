# P-LAB | Engineering & Software BIM

Repositório de soluções de automação para Engenharia Civil, focadas em alta performance e escalabilidade.

## 🛠 Tecnologias Utilizadas
- **Linguagens:** Python, C#
- **Frameworks:** FastAPI, pyRevit
- **BIM Core:** Autodesk Revit API, Autodesk Platform Services (APS), IfcOpenShell
- **DevOps:** GitHub, Headless BIM

## 🚀 Diferenciais Técnicos
- **Headless BIM:** Edição de arquivos IFC e extração de dados sem abertura de interface nativa.
- **Data Intelligence:** Integração de tabelas Revit com Excel via COM (Component Object Model).
