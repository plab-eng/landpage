# P-LAB | Picanço Logic Automation Build
> **Precision in Code, Excellence in Build.**

Repositório oficial de automações e plugins para a indústria AEC (Architecture, Engineering, and Construction). Desenvolvido por **José Picanço**, Engenheiro Civil e entusiasta de BIM.

## 2. Soluções Disponíveis
* **Export Schedules:** Automação de extração de tabelas do Revit para Excel via COM.
* **Export Sheets:** Gestão e exportação em massa de pranchas (PDF/DWG/DWF) com organização automática.

## 3. Requisitos
Nossos plugins rodam sobre a engine **pyRevit**. 
* **Nota:** O instalador oficial da P-LAB já configura automaticamente o **pyRevit** e o **Python** caso você não os tenha instalado.

## 4. Tecnologias Utilizadas
* Python (pyRevit Framework)
* Autodesk Revit API
* C# (para wrappers de instalação)

## 🚀 Diferenciais Técnicos
- **Headless BIM:** Edição de arquivos IFC e extração de dados sem abertura de interface nativa.
- **Data Intelligence:** Integração de tabelas Revit com Excel via COM (Component Object Model).
